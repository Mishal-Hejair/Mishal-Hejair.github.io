About ATOM

There are many features in ATOM like packages which will help make your life both easier and simpiler.

For Example:

  • autoclose-html-plus: it will help you automatically close HTML tags it was devloped by: binaryfunt
  • csslint: it adds css error checking abilities while using ATOM it was devloped by: tcarlsen
  • emmet: it adds code expansion it was devloped by: emmetio
  • atom-beautify: it helps you automatically format your code in atom it was devloped by: Glavin001
  • atom-html-preview: it helps you preview your website from ATOM and edit from there which makes your life easier it was devloped by: harmsk
    • and many more.

ATOM has cross-platform editing, smart autocompletion which can help you write your code faster, file system browser which will help you browse and open a file and a whole project in one window and many more.
