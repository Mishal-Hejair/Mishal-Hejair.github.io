How to use each CSS code correctly:

  • border = border is the property that allow you to specify the style, width,and color of an elements's border.
  • font = font has a lot of feature like font-style amd font-family and many more each one has a diffrent feature say for example you want to change the text font you write font-family and write the font like italic.
  • background = background is a feature that can let you change the color of your background or keep un image of youur background.
  • color = color is when you want to change the text color of your paragraph or title.
  • padding = padding is when you wanna kepp space for your table and boxes and many more.
  • margin = marging is the same as padding but simpiler,say for example you want to shift all your work to the right you just write margin-right or top or bottom and so on and it shifts it all.
  • float = float is basically pushing them all to a direction.
  • Height = height is how tall you want it to be
  • Width = width is changing the width of your image or table .
  • text-align = text align is basically making your text in the center or right or left.
