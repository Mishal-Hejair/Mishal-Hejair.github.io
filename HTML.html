How to use each HTML code correctly:

  • HTML = in every start of your html code you need to write HTML "lang" is language and "en" means english dir= "ltr" means left to right text direction so basically your telling the code how your going to start and what language and from where do you want to start.
  • TITLE = title is what you want to name your browser link.
  • BODY = body is where you write all your code, so basically your content.
  • h1 = h1 is basically heading 1 each heading becomes smallier and smallier and its the title in the content page of your website.
  • p = p stands for paragraph its basically what you want to write under your h1,h2,h3 its where you write your long words.
  • IMG = img is a tag that defines a photo un image to keep un image at your website.
  • UL = ul is a tag that defines an ordered bullet point list.
  • LI = li is a tag that defines a list of stuff you want to write.
  • TABLES = Tables is making a table in your website.
  • B = b is a tag that can make your text bold.
